# Contributing

Thanks for checking this out! Since it's just me working on this, I'm happy to have any extra sets of eyes or hands.

## How to contribute
* **Found a bug?** Open an [issue](https://github.com/KKDad/ComicCacher/issues) and tell me what happened.
* **Have a feature idea?** Open an issue first so we can chat about it before you spend time coding.
* **Want to fix something?** Fork the repo, make a branch, and send over a Pull Request.

## Rules of the road
* Keep it clean and be nice. Check out the [Code of Conduct](./CODE_OF_CONDUCT.md) for details.
* Try to keep your commits clear so I can actually tell what you changed.

That’s pretty much it. Thanks for helping out!
