== Cache Operations REST Service

The provides the entry point into the cache service.
- update the cache for all configured comics
- update the cache for a specific comic

=== Update cache for all comics

Verify that all comics have been cached and are ready to be served up. Comics are automatically cached daily, however
this method can be called to ensure that if any failed, they are retried.

This method will automatically go back the maximum # of days allowed, typically 7 days.
